Mõiste valdkond „Contsys.org“ (contsys.org)

  • attestation information base component documenting the details of an attestation of a set of electronic health record components
  • audit information documentation of the details of a revision of a base component
  • base component abstract class that include the properties common for all revisable classes included in EHR extract
  • composition structure component containing information composed during one clinical encounter or documentation session, and committed within one EHR
  • demographic extract demographic information extracted for the purpose of being included in an EHR extract or an EHR audit log extract
  • extracted component set structure component extracted from an EHR for the purpose of being communicated in an EHR extract as the target of one or more links
  • folder structure component used for creating hierarchical structures within an EHR extract and with compositions as leaf nodes
  • link base component that describes an association from information in an EHR extract to other information within that EHR extract
  • structure component electronic health record component which contain one of more other electronic health record components
