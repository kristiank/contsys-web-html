Mõiste „tervishoiuprotsess“ (healthcare process)

Tervishoiuprotsess

Termin: tervishoiuprotsess (healthcare process ⧉)
Määratlus: omavahel seotud või vastastikku mõjutavad tervishoiutegevused, mis muudavad sisendid väljunditeks (set of interrelated or interacting healthcare activities which transforms inputs into outputs)
Valdkond: protsess (process)
Märkus 1: The main type of healthcare process is the clinical process that has a health state as input and output and includes all activities in relation to one or more specified health issues.
(The main type of healthcare process is the clinical process that has a health state as input and output and includes all activities in relation to one or more specified health issues.)
Märkus 2: A healthcare process is not by definition restricted to one healthcare provider or any other organizational unit borders.
(A healthcare process is not by definition restricted to one healthcare provider or any other organizational unit borders.)
